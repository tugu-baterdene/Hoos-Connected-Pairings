# Hoos-Connected-Pairings

## Overview
Hoos Connected @ UVA has historically had issues with with its pairing assignments——it is an arduous process for the Program Coordinator to balance every students schedule, especially with the sporadic schedule changes that occur as the semester starts. 

This is a proposed automated pairing algorithm for Hoos Connected so that the process is streamlined. It will automatically take any students' schedule changes and account for that in it's final pairings.
